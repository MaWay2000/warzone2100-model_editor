# Warzone 2100 PIE Model Viewer

A small browser tool for inspecting and editing Warzone 2100 `.pie` models.

## Features

- Loads `PIE 2` and `PIE 3` files
- Supports multi-level PIE models
- Renders geometry in a Three.js viewport
- Auto-binds uploaded texture pages by filename
- Shows points, wireframe, connectors, and model metadata
- Lets you edit vertex coordinates and export the updated `.pie`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
